✓ Our Solution

Dynamic Signals,
Driven by Data

FlowMind replaces fixed clocks with a weight-based algorithm. Every second, the system measures real vehicle density per lane and assigns green time proportionally.

1
YOLOv8 Vehicle Detection
Live camera feeds processed by YOLOv8 count cars, trucks, and buses in real-time with class-weighted scoring.
2
Weight Calculation
W = Cars×1 + Trucks×1.5 + Buses×2.5. Heavier vehicles earn priority. The heaviest lane gets green first.
3
Adaptive Green Duration
Green time = clamp(6, 18, Weight × 1.2). Dense lanes get up to 3× more clearance time than sparse ones.
4
Emergency Corridor Override
When an emergency vehicle is detected, all lanes instantly red and the corridor opens within one tick.
function computeWeight(lane) {
  return
    lane.cars   × 1.0 // standard car
  + lane.trucks × 1.5 // large vehicle
  + lane.buses  × 2.5; // max weight
}
 
function greenDuration(weight) {
  return clamp(
    6,        // min 6s
    18,      // max 18s
    weight × 1.2 // adaptive
  );
}
